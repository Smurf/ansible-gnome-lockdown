Smurf.Gnome_Lockdown
====================

Collection version 1.0.0

.. contents::
   :local:
   :depth: 1

Description
-----------

Use ansible to manage and lock GSettings in Gnome.

**Author:**

* Sam Coles <me@smurf.codes>

**Supported ansible-core versions:**

* 2.9.10 or newer
