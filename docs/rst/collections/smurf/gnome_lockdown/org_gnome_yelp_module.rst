smurf.gnome_lockdown.org_gnome_yelp module -- Manages GNOME GSettings for org.gnome.yelp
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

Synopsis

.. Description

- This module allows for the configuration of GSettings keys within the 'org.gnome.yelp' schema.
- It supports setting default values and locking keys to enforce system\-wide policies.

        show-cursor_locked: true

        font-adjustment_locked: true
